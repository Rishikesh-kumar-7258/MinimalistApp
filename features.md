# Minimalist Launcher — Feature List

## Core Experience
1. No wallpaper — solid or subtle gradient background only
2. No app icons — text-only interface throughout
3. Unified search bar — searches apps, contacts, and settings in one place
4. Minimalist typography-first design
5. Customizable color scheme (background + text + accent colors)
6. App drawer — scrollable, text-based list of all installed apps
7. App usage tracking with per-app time limits and restrictions

---

## Home Screen
8. Pinnable text shortcuts — pin up to 5 apps or contacts on the home screen as plain text
9. Minimal text widgets — clock, date, weather, and next calendar event as plain text lines (no icons, no cards)
10. Contextual app suggestions — surface 2–3 relevant apps based on time of day and usage patterns

---

## Organization
11. Text-based app groups — label apps into categories (Work, Social, Tools, etc.) accessible as collapsible text sections
12. Frequency-based sorting — optionally sort the app drawer by most-used
13. Hidden apps — hide apps from the drawer without uninstalling

---

## Focus & Digital Wellbeing
14. Focus profiles — Work, Personal, Sleep, and Custom profiles that restrict or allow specific apps
15. Profile scheduling — automatically switch profiles on a time schedule (e.g., Work profile 9am–6pm)
16. Per-app time windows — allow an app only within a defined daily time slot (e.g., Reddit 8pm–9pm)
17. Friction screen — a brief mandatory pause (3–5 seconds) before opening designated apps, replacing intrusive popups
18. Daily usage report — clean text summary of screen time per app, checked at a user-chosen time
19. Weekly trends — 7-day usage history with comparison to prior week
20. Usage streaks — track consecutive days under a set screen time goal for accountability

---

## Gestures & Navigation
21. Configurable swipe gestures — assign actions to swipe up, down, left, right, and double-tap on the home screen (e.g., swipe down opens dialer)
22. Pull-down search — pull down anywhere on the home screen to open the search bar
23. Long-press context menu — long-press any app name to pin, hide, group, or set restrictions

---

## Customization
24. Font selection — choose from a small set of clean system and bundled fonts
25. Font size — small / medium / large
26. Text alignment — left, center, or right for home screen elements
27. Theme modes — Light, Dark, AMOLED black (for battery saving on OLED screens)
28. Clock format — 12h or 24h, with optional seconds display

---

## Utilities
29. Quick scratch pad — a single tap to open a plain-text note directly from the home screen; saves automatically
30. App lock — lock specific apps behind PIN or biometric authentication
31. Emergency bypass — a whitelist of essential apps (dialer, messages, maps) that are never restricted

---

## Privacy & Data
32. Zero telemetry — no analytics, no crash reporting sent off-device
33. No internet permission — the launcher itself requires no network access; weather uses a user-configured API key stored locally
34. Backup & restore — export all settings and preferences to a local file; import on a new device

---

## Market Gaps This App Fills
- **No other minimalist launcher offers per-app time windows** (not just daily limits, but time-of-day slots)
- **Friction screens instead of hard blocks** — gentler than outright app blocking, more effective than popups
- **Text widgets with no icons** — every other launcher either removes widgets entirely or keeps icon-heavy cards
- **Local-only backup** — users lose all configuration after resetting; no app solves this without cloud sync
- **Gesture-first navigation** — current apps rely on taps; full swipe customization is absent in the minimalist space
- **Quick scratch pad** — users consistently return to full launchers just for note access; this removes that reason
